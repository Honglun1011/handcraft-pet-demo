# Handcraft Pet Custom Demo

A static clickable demo for a curated traditional handcraft pet custom website.

## Preview Locally

Open `index.html` directly, or run the included PowerShell static server:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\server.ps1
```

Then visit:

```text
http://127.0.0.1:4173/?v=2#home
```

## Demo Scope

- Product Gallery
- Craft Information
- 13-step Pet Custom Craft Brief Wizard
- Stories
- About
- Contact
- FAQ / Policy
- Admin Demo backed by browser `localStorage`

No payment, cart, auth, or production backend is included.
